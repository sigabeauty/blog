关于 SVM 需要了解的

继续经典机器学习算法的分享，这次我来分享的是支持向量机 SVM。

支持向量机（Support Vector Machine, SVM）是机器学习中经典的监督学习分类算法。其核心思想是在特征空间中寻找一个最优的分类超平面，使得不同类别的样本点到该超平面的最小距离最大化，从而获得较好的泛化能力。

问题引入：分类超平面与间隔

假设在二维平面上，给定一个包含 $N$ 个样本的二分类数据集：

$$D = {(x_1, y_1), (x_2, y_2), \dots, (x_N, y_N)}$$

其中，$x_i = [x_{i1}, x_{i2}]^T \in \mathbb{R}^2$ 表示样本点在平面上的坐标向量，$y_i \in {+1, -1}$ 为其对应的分类标签。

若该数据集在线性空间中是线性可分的，意味着存在无数条直线可以将两类点完全隔开。在 $d$ 维空间中，这条线被泛化为超平面（Hyperplane），其线性方程可表示为：

$$w^T x + b = 0$$

其中，$w = [w_1, w_2, \dots, w_d]^T$ 为超平面的法向量，决定了超平面的方向；$b$ 为截距，决定了超平面与原点的距离。

这里有人会问，为什么需要最大化间隔？普通的线性分类器（如感知机）只要找到任意一个满足 $y_i(w^T x_i + b) > 0$ 的超平面就会停止迭代。然而，若分类超平面紧贴着某类样本点，当测试数据包含轻微噪声时，样本点容易跨越分类线导致误判。SVM 的目标是找到位于两类数据正中间、且使两侧“安全缓冲区”（即间隔 Margin）最大化的超平面。

1. SVM 的数学建模

点到超平面的几何距离

在解析几何中，空间中任意一点 $x_i$ 到超平面 $w^T x + b = 0$ 的垂直几何距离 $\gamma_i$ 公式为：

$$\gamma_i = \frac{|w^T x_i + b|}{|w|}$$

其中 $|w| = \sqrt{w_1^2 + w_2^2 + \dots + w_d^2}$ 为法向量的 $\text{L}_2$ 范数。

为了消去绝对值符号，我们利用标签 $y_i \in {+1, -1}$ 的物理含义：

  • 若 $y_i = +1$，合理的分类应满足 $w^T x_i + b > 0$；
  • 若 $y_i = -1$，合理的分类应满足 $w^T x_i + b < 0$。

因此，只要分类正确，恒有 $y_i(w^T x_i + b) > 0$。由此几何距离可重写为：$$\gamma_i = \frac{y_i(w^T x_i + b)}{|w|}$$

缩放变换与约束条件简化

假设我们找到了某个超平面 $(w, b)$。如果我们将 $w$ 和 $b$ 等比例放大 $k$ 倍（变为 $kw$ 和 $kb$），超平面方程 $k w^T x + k b = 0$ 所代表的分类面完全没有改变。但此时分子 $y_i(w^T x_i + b)$ 也会扩大 $k$ 倍。

这种尺度的任意性给优化带来了困难。为了锁定唯一的 $w$ 和 $b$，SVM 引入了一个合理的约定：通过等比例缩放 $w$ 和 $b$，使得距离超平面最近的样本点（即支持向量），其分子部分恰好等于 1。即：

$$\min_{i=1,\dots,N} y_i(w^T x_i + b) = 1$$

既然最近的点的函数值被固定为 1，那么对于数据集中的所有样本点，必然满足以下不等式约束：

$$y_i(w^T x_i + b) \ge 1, \quad i = 1, 2, \dots, N$$

构建目标函数

根据上述约定，最近的样本点到超平面的几何距离为 $\gamma = \frac{1}{|w|}$。由于超平面居于两类数据的正中间，总的分类间隔（Margin）即为正负两边最近点到平面距离的和：

$$\text{Margin} = 2\gamma = \frac{2}{|w|}$$

我们的优化目标是最大化这个总间隔：

$$\max_{w, b} \frac{2}{|w|}$$

在数学上，最大化 $\frac{2}{|w|}$ 等价于最小化 $\frac{|w|}{2}$，为了后续求导时方便消去常数项，通常将其转化为最小化其平方的二分之一。由此，我们得到了硬间隔 SVM 的原始最优化问题（Primal Problem）：

$$\min_{w, b} \frac{1}{2}|w|^2$$ $$\text{s.t.} \quad 1 - y_i(w^T x_i + b) \le 0, \quad i = 1, 2, \dots, N$$

2. 数学求解：拉格朗日乘子法与对偶问题

因为原始目标函数带有 $N$ 个不等式约束，无法直接通过求导求解。我们需要将其转化为无约束的形式。

构建拉格朗日函数

引入拉格朗日乘子 $\alpha_i \ge 0 \ (i=1,2,\dots,N)$，将约束条件融入目标函数中，构建拉格朗日函数： $$L(w, b, \alpha) = \frac{1}{2}|w|^2 + \sum_{i=1}^N \alpha_i \Big( 1 - y_i(w^T x_i + b) \Big)$$ 将其展开改写，以便后续关于 $w$ 和 $b$ 的求导： $$L(w, b, \alpha) = \frac{1}{2}w^T w + \sum_{i=1}^N \alpha_i - \sum_{i=1}^N \alpha_i y_i w^T x_i - \sum_{i=1}^N \alpha_i y_i b$$

引入对偶性

原问题可以表达为 $\min_{w,b} \max_{\alpha \ge 0} L(w,b,\alpha)$。根据凸优化理论，由于目标函数 $\frac{1}{2}|w|^2$ 是凸函数，且约束条件是关于 $w$ 和 $b$ 的线性不等式，该问题满足 Slater 条件，因此强对偶性（Strong Duality）成立。

强对偶性允许我们交换 $\min$ 和 $\max$ 的顺序，将原问题转化为等价的对偶问题（Dual Problem）： $$\max_{\alpha \ge 0} \min_{w, b} L(w, b, \alpha)$$

这意味着我们可以先固定 $\alpha$，对内层的 $w$ 和 $b$ 求最小值，再对外层求关于 $\alpha$ 的极大值。

内层消元（对 $w$ 和 $b$ 求偏导）

为了求内层 $\min_{w, b} L(w, b, \alpha)$，我们分别对 $w$ 和 $b$ 计算偏导数，并令其等于 0：

对 $w$ 求偏导： $$\frac{\partial L}{\partial w} = w - \sum_{i=1}^N \alpha_i y_i x_i = 0 \implies w = \sum_{i=1}^N \alpha_i y_i x_i \quad \text{（等式 ①）}$$

对 $b$ 求偏导： $$\frac{\partial L}{\partial b} = -\sum_{i=1}^N \alpha_i y_i = 0 \implies \sum_{i=1}^N \alpha_i y_i = 0 \quad \text{（等式 ②）}$$

代回

现在我们需要将等式 ① 和等式 ② 代回拉格朗日函数中，消去 $w$ 和 $b$。请看详细步骤：

首先，将展开的 $L(w, b, \alpha)$ 重新组合，提出公因式：

$$L(w, b, \alpha) = \frac{1}{2}w^T w - w^T \left( \sum_{i=1}^N \alpha_i y_i x_i \right) - b \left( \sum_{i=1}^N \alpha_i y_i \right) + \sum_{i=1}^N \alpha_i$$

处理含有 $b$ 的项：因为等式 ② 确定了 $\sum_{i=1}^N \alpha_i y_i = 0$，所以 $-b \left( \sum_{i=1}^N \alpha_i y_i \right) = -b \cdot 0 = 0$。此项直接消除。

处理含有 $w$ 的项：将等式 ① 的 $w = \sum_{i=1}^N \alpha_i y_i x_i$ 代入前两项：

$$L(w, b, \alpha) = \frac{1}{2}w^T w - w^T w + \sum_{i=1}^N \alpha_i = -\frac{1}{2}w^T w + \sum_{i=1}^N \alpha_i$$

完全展开 $w^T w$：将 $w = \sum_{i=1}^N \alpha_i y_i x_i$ 代入 $w^T w$，根据多项式乘法，需引入双重循环下标 $i$ 和 $j$： $$w^T w = \left( \sum_{i=1}^N \alpha_i y_i x_i \right)^T \left( \sum_{j=1}^N \alpha_j y_j x_j \right) = \sum_{i=1}^N \sum_{j=1}^N \alpha_i \alpha_j y_i y_j (x_i^T x_j)$$

将这一结果代回，我们得到了仅包含 $\alpha$ 的函数： $$\min_{w, b} L(w, b, \alpha) = \sum_{i=1}^N \alpha_i - \frac{1}{2} \sum_{i=1}^N \sum_{j=1}^N \alpha_i \alpha_j y_i y_j (x_i^T x_j)$$

最终标准形式

加上外层的 $\max_{\alpha}$ 以及对拉格朗日乘子自身的约束，最终的对偶问题可写为： $$\max_{\alpha} \sum_{i=1}^N \alpha_i - \frac{1}{2} \sum_{i=1}^N \sum_{j=1}^N \alpha_i \alpha_j y_i y_j (x_i^T x_j)$$ $$\text{s.t.} \quad \sum_{i=1}^N \alpha_i y_i = 0$$ $$\alpha_i \ge 0, \quad i = 1, 2, \dots, N$$

3. KKT 条件 （怎么确定支持向量）

在求解带不等式约束的拉格朗日问题时，最优解必须满足 KKT（Karush-Kuhn-Tucker）条件。在本问题中，最核心的是互补松弛条件（Complementary Slackness）： $$\alpha_i \Big( 1 - y_i(w^T x_i + b) \Big) = 0$$

这一条件意味着，对于每一个样本点 $x_i$，只有以下两种情况之一能成立：

  1. $\alpha_i = 0$：此时 $1 - y_i(w^T x_i + b) < 0 \implies y_i(w^T x_i + b) > 1$。表明这些样本点处于缓冲区外部，它们对最终超平面参数 $w$ 和 $b$ 的计算没有任何贡献。

  2. $\alpha_i > 0$：为了让乘积为 0，必须有 $1 - y_i(w^T x_i + b) = 0 \implies y_i(w^T x_i + b) = 1$。表明这些样本点恰好落在缓冲区的边界上。

这些满足 $\alpha_i > 0$ 的点就是支持向量（Support Vectors）。最终的法向量 $w = \sum_{i=1}^N \alpha_i y_i x_i$ 实际上只由这少数几个支持向量决定。

4. 从线性可分到线性不可分

现实世界中分类平面的点并不总是完美线性可分的。随着点集复杂度的上升，会出现线性不可分的情况。

完全线性可分 $\rightarrow$ 硬间隔 SVM

这类数据没有任何噪声，两类点之间有明显的空白带。通过二次规划算法解出最优的 $\alpha_i$。

  1. 利用 $w = \sum_{i=1}^N \alpha_i y_i x_i$ 算出 $w$。
  2. 选取任意一个满足 $\alpha_s > 0$ 的支持向量 $x_s$，由于 $y_s(w^T x_s + b) = 1$，我们在等式两边同乘 $y_s$（注意 $y_s^2 = 1$），可得：

$$b = y_s - w^T x_s$$

  1. 决策函数为：$f(x) = \text{sign}(w^T x + b)$。

混入轻微噪声 $\rightarrow$ 软间隔 SVM

这种情况下两类点整体上依然可以通过一条直线分开，但是局部由于测量误差等原因，混入了部分交叉错位的样本点。如果强行用硬间隔去切分，会导致超平面极其扭曲（过拟合），甚至由于约束冲突而无解。

对着这类问题，我们通过以下两种方法改造：

  1. 引入松弛变量 $\xi_i$：我们允许样本点适度偏离其合法的边界。允许偏离的几何距离称为松弛变量 $\xi_i \ge 0$。此时约束条件放宽为：

$$y_i(w^T x_i + b) \ge 1 - \xi_i$$

  1. 加入惩罚项：为了防止松弛变量无限放大，必须在目标函数中对所有的偏离距离进行求和惩罚，并引入惩罚参数 $C > 0$。新目标函数变为：

$$\min_{w, b, \xi} \frac{1}{2}|w|^2 + C \sum_{i=1}^N \xi_i$$

经过同样的拉格朗日求导推导（只需增加对 $\xi_i$ 的求偏导），其对偶问题与硬间隔高度一致，唯一的改变是拉格朗日乘子 $\alpha_i$ 被加上了上限 $C$：

$$\max_{\alpha} \sum_{i=1}^N \alpha_i - \frac{1}{2} \sum_{i=1}^N \sum_{j=1}^N \alpha_i \alpha_j y_i y_j (x_i^T x_j)$$ $$\text{s.t.} \quad \sum_{i=1}^N \alpha_i y_i = 0, \quad 0 \le \alpha_i \le C, \quad i = 1, 2, \dots, N$$

当 $C \to \infty$ 时，模型对错误零容忍，退化为硬间隔；当 $C$ 较小时，模型允许少量点跨界，以换取更平直、泛化能力更强的分类线。

完全线性不可分 $\rightarrow$ 核函数 SVM

对于更复杂的数据分布，平面上的点呈现出同心圆或靶盘状分布，内圈是正类，外圈是负类。在二维空间里，没有任何一条直线能将它们分开。

由于低维不可分，可以考虑将原本平躺在二维平面上的点 $x$，通过一个映射函数 $\phi(x)$ 投射到三维空间。例如，通过增加一个高度特征（如点到原点的距离平方），内圈的点就会由于距离小而被映射在半空中，外圈的点留在桌面上。此时，我们就可以在三维空间中用一个平整的超平面切开它们。

如果直接将点升到高维，我们在对偶问题中就需要计算高维向量的内积 $\phi(x_i)^T \phi(x_j)$。如果高维空间的维度是一万维甚至无穷维，这种直接计算在计算机内是无法实现的。

核方法（Kernel Trick）

回顾对偶问题，样本点只以两两内积 $(x_i^T x_j)$ 的形式存在于公式中。因此，我们不需要显式地写出高维映射 $\phi(x)$ 的具体形式，而是寻找一个特殊的低维函数 $K(x_i, x_j)$，使得：

$$K(x_i, x_j) = \phi(x_i)^T \phi(x_j)$$

也就是说，我们直接在低维空间传入 $x_i$ 和 $x_j$ 计算得到一个标量，而这个标量在数值上等价于它们升到高维空间后的内积。以最常用的高斯核函数（Radial Basis Function, RBF）为例：

$$K(x_i, x_j) = \exp\left( -\gamma |x_i - x_j|^2 \right)$$

在实际操作中，我们只需将对偶优化目标以及决策函数中的所有内积项 $(x_i^T x_j)$ 替换为 $K(x_i, x_j)$，即可在不增加计算开销的前提下，隐式地在无穷维空间中完成线性切分：

$$f(x) = \text{sign}\left( \sum_{i=1}^N \alpha_i y_i K(x_i, x) + b \right)$$

支持向量机通过“最大化间隔”这一简单且优雅的几何准则，结合对偶理论与核技巧，有效地解决了高维非线性分类问题。尽管现在开始出现一些基于深度神经网络的方法在处理大规模数据时很有效，但 SVM 更适合在小样本的高维度场景下使用。
